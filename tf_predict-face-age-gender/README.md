# tf_predict-face-age-gender
facenetのMTCNNで顔検出して、年齢・性別を学習済みのWideResNetで予測する
- https://github.com/ipazc/mtcnn と https://github.com/yu4u/age-gender-estimation/ のパクリ
- 顔検出は精度よく検出できるが、年齢・性別予測は精度全然ダメ

## Usage
run_predict.ipynb 参照
